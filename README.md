# ads-assignemnt3-part1

In this assignment you are to:

1. **(0 points)** Pick a variant using your student ID as a random seed. For that you need to update and run `VariantPicker.main()` method.
2. **(3 points)** Implement a generic elementary sorting algorithm to sort an array of `Comparable` objects according to their natural order.
3. **(3 points)** Implement a generic elementary sorting algorithm to sort an array of objects using a custom `Comparator`.
4. **(2 points)** Implement a "natural" sorting order for `Person` class according to your variant by implementing `Comparable<Person>` interface.
5. **(2 points)** Implement a "custom" sorting order for `Person` class according to your variant by implementing `Comparator<Person>` interface.


Your solutions will be run against a number of tests for auto-grading.

Best of Luck!
